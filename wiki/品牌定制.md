# 品牌定制

Auth9 提供全面的品牌定制功能，让您可以自定义租户的视觉外观，包括 Logo、颜色、主题等，打造专属的品牌体验。

## 功能概述

- ✅ 自定义租户 Logo
- ✅ 自定义主题颜色
- ✅ 自定义 Favicon
- ✅ 自定义登录页面外观
- ✅ 自定义邮件模板样式
- ✅ 租户级别独立配置
- ✅ 实时预览效果

## 品牌元素

### 1. Logo 配置

每个租户可以配置以下 Logo：

| Logo 类型 | 尺寸建议 | 使用场景 |
|----------|---------|---------|
| **主 Logo** | 200x60 px | Portal 顶部导航栏 |
| **小 Logo** | 40x40 px | 移动端、Favicon |
| **登录页 Logo** | 300x100 px | Keycloak 登录页面 |
| **邮件 Logo** | 200x60 px | 邮件头部 |

**支持格式**: PNG, SVG, JPEG, WebP

### 2. 颜色主题

可以自定义的颜色包括：

| 颜色名称 | 用途 | 默认值 |
|---------|------|-------|
| **Primary Color** | 主要按钮、链接、高亮 | `#007AFF` |
| **Secondary Color** | 次要按钮、辅助元素 | `#5856D6` |
| **Accent Color** | 强调色、提示 | `#FF9500` |
| **Background Color** | 页面背景 | `#FFFFFF` |
| **Text Color** | 主要文字 | `#1C1C1E` |
| **Border Color** | 边框、分隔线 | `#E5E5EA` |

### 3. 字体设置

支持自定义字体族：

```json
{
  "font_family": "Inter, -apple-system, BlinkMacSystemFont, sans-serif",
  "heading_font": "SF Pro Display, Inter, sans-serif"
}
```

**内置字体选项**:
- Inter (默认)
- SF Pro (苹果风格)
- Roboto (Material Design)
- 思源黑体 (Noto Sans CJK)

## 配置品牌

### 通过管理界面配置

1. 登录 Auth9 Portal
2. 导航到 **Settings** > **Branding**
3. 选择目标租户
4. 配置品牌元素：

#### Logo 上传

1. 在 "Logos" 区域点击对应的上传按钮
2. 选择图片文件（最大 2MB）
3. 系统自动裁剪和优化
4. 点击 **Save** 保存

**提示**: 建议使用透明背景的 PNG 或 SVG 格式

#### 颜色主题

1. 在 "Colors" 区域点击颜色选择器
2. 使用色轮或输入 HEX 颜色码
3. 查看右侧实时预览
4. 点击 **Apply Theme** 应用

**预设主题**:
- 💙 **Auth9 Blue** (默认)
- 🟣 **Purple Dream**
- 🟢 **Nature Green**
- 🔴 **Passionate Red**
- ⚫ **Dark Mode**

#### 自定义 CSS

高级用户可以添加自定义 CSS：

```css
/* 自定义按钮样式 */
.btn-primary {
  border-radius: 12px;
  box-shadow: 0 4px 12px rgba(0, 122, 255, 0.3);
}

/* 自定义卡片样式 */
.card {
  backdrop-filter: blur(20px);
  background: rgba(255, 255, 255, 0.8);
}
```

### 通过 REST API 配置

#### 获取品牌配置

```bash
curl https://api.auth9.yourdomain.com/api/v1/tenants/{tenant_id}/branding \
  -H "Authorization: Bearer <token>"
```

响应：

```json
{
  "data": {
    "tenant_id": "tenant-uuid",
    "logo_url": "https://cdn.auth9.com/logos/tenant-uuid/logo.png",
    "small_logo_url": "https://cdn.auth9.com/logos/tenant-uuid/logo-sm.png",
    "favicon_url": "https://cdn.auth9.com/logos/tenant-uuid/favicon.ico",
    "primary_color": "#007AFF",
    "secondary_color": "#5856D6",
    "accent_color": "#FF9500",
    "background_color": "#FFFFFF",
    "text_color": "#1C1C1E",
    "border_color": "#E5E5EA",
    "font_family": "Inter, sans-serif",
    "custom_css": "",
    "updated_at": "2024-01-01T00:00:00Z"
  }
}
```

#### 更新品牌配置

```bash
curl -X PUT https://api.auth9.yourdomain.com/api/v1/tenants/{tenant_id}/branding \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "primary_color": "#007AFF",
    "secondary_color": "#5856D6",
    "logo_url": "https://example.com/logo.png",
    "font_family": "Inter, sans-serif"
  }'
```

#### 上传 Logo

```bash
curl -X POST https://api.auth9.yourdomain.com/api/v1/tenants/{tenant_id}/branding/logo \
  -H "Authorization: Bearer <token>" \
  -F "logo=@/path/to/logo.png" \
  -F "type=main"
```

支持的 `type` 参数：
- `main`: 主 Logo
- `small`: 小 Logo
- `favicon`: Favicon

#### 重置为默认主题

```bash
curl -X POST https://api.auth9.yourdomain.com/api/v1/tenants/{tenant_id}/branding/reset \
  -H "Authorization: Bearer <token>"
```

## 品牌资源管理

### 资源存储

品牌资源（Logo、图片等）存储选项：

1. **本地存储** (默认)
   - 存储在服务器本地文件系统
   - 通过 auth9-core 提供静态文件服务
   - 适合小规模部署

2. **对象存储**（推荐）
   - 支持 AWS S3、阿里云 OSS、腾讯云 COS
   - CDN 加速访问
   - 适合生产环境

配置示例：

```yaml
# config.yaml
storage:
  type: "s3"  # 或 "oss", "cos", "local"
  bucket: "auth9-assets"
  region: "us-west-2"
  cdn_domain: "cdn.auth9.yourdomain.com"
```

### 资源优化

系统会自动优化上传的品牌资源：

- ✅ 图片压缩（保持质量）
- ✅ 生成多种尺寸（响应式）
- ✅ 转换为 WebP 格式
- ✅ 添加缓存头（1年）

### 资源清理

删除租户时，相关的品牌资源会被自动清理。

也可以手动清理未使用的资源：

```bash
curl -X POST https://api.auth9.yourdomain.com/api/v1/admin/branding/cleanup \
  -H "Authorization: Bearer <admin-token>"
```

## 设计指南

### Logo 设计建议

1. **简洁明了**
   - 避免过于复杂的图形
   - 在小尺寸下也能清晰识别

2. **适配场景**
   - 提供深色和浅色背景的版本
   - 使用矢量格式（SVG）确保缩放不失真

3. **品牌一致性**
   - 与公司整体品牌保持一致
   - 使用相同的颜色和字体

### 颜色选择建议

1. **对比度**
   - 确保文字和背景有足够对比度（WCAG AA 标准）
   - 使用工具检查：[WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)

2. **色彩心理学**
   - 蓝色：信任、专业（金融、科技）
   - 绿色：成长、安全（环保、健康）
   - 红色：激情、紧迫（娱乐、销售）
   - 紫色：创新、奢华（艺术、高端）

3. **无障碍**
   - 不仅依靠颜色传达信息
   - 考虑色盲用户（红绿色盲最常见）

### 响应式设计

品牌元素会自动适配不同屏幕尺寸：

| 屏幕尺寸 | Logo 尺寸 | 字体大小 |
|---------|----------|---------|
| 手机 (< 768px) | 小 Logo | 14px |
| 平板 (768-1024px) | 中等 Logo | 16px |
| 桌面 (> 1024px) | 完整 Logo | 16px |

## 高级定制

### 自定义域名

为租户配置独立域名：

```json
{
  "custom_domain": "auth.mycompany.com",
  "ssl_enabled": true
}
```

需要配置 DNS CNAME 记录指向 Auth9 服务器。

### 白标模式

完全隐藏 Auth9 品牌，仅显示租户品牌：

```json
{
  "white_label": true,
  "hide_powered_by": true,
  "custom_footer": "© 2026 My Company. All rights reserved."
}
```

**注意**: 白标模式可能需要企业版授权。

### 多语言支持

品牌配置支持多语言文本：

```json
{
  "company_name": {
    "en": "My Company",
    "zh": "我的公司",
    "ja": "私の会社"
  },
  "tagline": {
    "en": "Your trusted partner",
    "zh": "您值得信赖的伙伴"
  }
}
```

## 常见问题

### Logo 显示模糊？

1. 使用更高分辨率的图片（2x 或 3x）
2. 使用 SVG 矢量格式
3. 检查图片压缩质量设置

### 颜色在不同设备上显示不一致？

这是正常的色彩管理差异。建议：
1. 使用 sRGB 色彩空间
2. 在多种设备上测试
3. 避免使用极端饱和的颜色

### 如何导出当前品牌配置？

```bash
curl https://api.auth9.yourdomain.com/api/v1/tenants/{tenant_id}/branding/export \
  -H "Authorization: Bearer <token>" \
  -o branding-config.json
```

### 如何批量配置多个租户？

使用品牌模板功能：

1. 创建品牌模板
2. 应用模板到多个租户

```bash
curl -X POST https://api.auth9.yourdomain.com/api/v1/branding/templates/{template_id}/apply \
  -H "Authorization: Bearer <token>" \
  -d '{"tenant_ids": ["tenant-1", "tenant-2", "tenant-3"]}'
```
