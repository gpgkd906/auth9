# Auth9 Wiki 主页

欢迎来到 Auth9 项目 Wiki！

## 📖 关于 Auth9

Auth9 是一个自托管的身份认证和访问管理服务，旨在替代昂贵的商业解决方案（如 Auth0）。它提供了企业级的身份管理、单点登录（SSO）和动态基于角色的访问控制（RBAC）功能。

## 🚀 核心特性

- **多租户架构** - 完整的租户隔离和独立配置
- **B2B 自助入驻** - 企业用户自助注册并创建组织
- **单点登录（SSO）** - 基于 OIDC 协议的企业级 SSO
- **企业 SSO 连接器** - 租户级别的 OIDC/SAML 身份提供商配置
- **动态 RBAC** - 灵活的角色权限管理，支持角色继承
- **Token 交换** - 服务间认证和授权的高性能实现
- **审计日志** - 完整的操作审计追踪
- **现代化 UI** - 基于 React Router 7 的管理界面
- **Action Engine** - 事件驱动的自动化工作流系统
- **TypeScript SDK** - 官方 SDK，简化集成开发
- **邀请系统** - 邮件邀请用户加入租户，自动化入职流程
- **品牌定制** - 自定义 Logo、颜色、主题等品牌元素
- **邮件模板** - 灵活的邮件模板系统，支持多语言
- **Keycloak 主题** - 自定义认证界面主题
- **密码管理** - 密码策略、重置和修改功能
- **会话管理** - 查看和撤销活跃会话
- **WebAuthn 支持** - Passkey 无密码认证
- **社交登录** - 支持 Google、GitHub、OIDC、SAML
- **安全告警** - 实时威胁检测和告警
- **登录分析** - 详细的登录统计和事件日志
- **Webhook** - 实时事件通知机制
